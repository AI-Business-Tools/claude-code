# Examples

Sample outputs demonstrating what the skills in this repository produce. These are included so you can see the end result before installing a skill.

## Available Examples

### [Slides from Paper](slides-from-paper/)

Demonstrates the `slides-content` pipeline: an academic paper processed through the split-read-summarize-compile workflow, producing a Beamer slide deck. The example shows the output structure, slide types, and quality gate results without the source paper (which is not redistributable).
